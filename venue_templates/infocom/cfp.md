## Page Limit & Format

- **Total length:** Maximum 10 printed pages
- **Main text** (including figures, tables, appendices): 9 pages
- **References:** Excluded from the 9-page limit
- IEEE two-column format; IEEEtran.cls v1.8
- Preamble: `\documentclass[10pt, conference, letterpaper]{IEEEtran}`
- Font: 10pt Times, two-column, US letter size
- Submit in PDF format via EDAS

## Topics

AI/ML for networking, 6G and beyond cellular systems, Big data & data processing, Cellular networks, Cloud computing, Cyber-physical systems, Datacenter networking, Energy efficiency, Fault tolerance/reliability/survivability, Information security & privacy, Interference management, IoT, Mobile networks, Network slicing, Optical networks, P2P, QoS, Routing & traffic engineering, SDN/NFV, Wireless networks.

## Anonymity

Double-blind. Remove author names/affiliations, acknowledgements identifying names/funding, project names traceable via web search. Remove PDF metadata. Papers not conforming → rejected without review.

## Other Policies

- Abstract (up to 200 words) required as EDAS metadata
- Submissions via EDAS
- Maximum 5 papers per author
