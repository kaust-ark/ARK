# ICLR 2025 — Call for Papers

**Source:** https://iclr.cc/Conferences/2025/CallForPapers
**Conference Dates:** April 24–28, 2025 (Sessions: Apr 24–26; Workshops: Apr 27–28)

## Key Dates

| Event | Date (AoE) |
|---|---|
| Abstract submission | September 27, 2024 |
| Full paper submission | October 1, 2024 |
| Reviews released | November 12, 2024 |
| Discussion period | November 12–26, 2024 |
| Decision notifications | January 22, 2025 |
| Camera-ready | March 14, 2025 |

## Page Limit & Format

- **Main text:** 6–10 pages (strictly enforced; page 11+ → desk rejection)
- References, ethics/reproducibility statements, acknowledgments: excluded
- Appendix: unlimited (reviewers not required to read)
- Template: https://github.com/ICLR/Master-Template/raw/master/iclr2025.zip
- Submit via OpenReview

## Topics

Representation learning across: Core ML (unsupervised/supervised/self-supervised learning, transfer, meta-learning, continual learning, RL, generative models, optimization, theory, probabilistic methods, causal reasoning, graphs/geometries), Applications (vision, audio, speech, language, robotics, neuroscience, physical sciences, healthcare, sustainability), Responsible AI (fairness, safety, privacy, interpretability, datasets/benchmarks).

## Anonymity

Double-blind. Fully anonymized — no author names, affiliations, identifying info. arXiv submissions permitted. Supplementary code must be anonymous.

## Other Policies

- Authors submitting ≥3 papers must register as reviewers (≥6 papers to review)
- LLMs permitted as writing assist; LLMs not eligible for authorship
- All authors must adhere to ICLR Code of Ethics
