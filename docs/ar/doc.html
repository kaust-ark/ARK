توثيق ARK

هذه الصفحة تقدم نسخة عربية مختصرة من التوثيق. للاطلاع على التوثيق الكامل يمكنك الرجوع إلى النسخة الإنجليزية أو الصينية.

التثبيت

git clone https://github.com/kaust-ark/ARK.git
cd ARK
pip install -e .

لتفعيل ميزات Deep Research:

pip install -e ".[research]"

أوامر أساسية

  • ark new <project-name> إنشاء مشروع جديد عبر wizard تفاعلي
  • ark run <project-name> تشغيل Pipeline
  • ark status <project-name> عرض الحالة الحالية
  • ark monitor <project-name> متابعة السجلات مباشرة
  • ark stop <project-name> إيقاف التشغيل بشكل آمن

إعداد المشروع

أهم الإعدادات في config.yaml:

code_dir: /path/to/your/code
venue: NeurIPS
model: claude
paper_accept_threshold: 8
telegram_bot_token: "YOUR_BOT_TOKEN"
telegram_chat_id: "YOUR_CHAT_ID"
